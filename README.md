## 🎬 Netflix Movies and TV Shoes Data Analysis using SQL.

Netflix SQL Project analyzes Netflix Movies &amp; TV Shows data using MySQL. It includes data cleaning, filtering, aggregations, CTEs, window functions, and business insights such as content trends, top countries, ratings, genres, directors, and yearly releases. Built to strengthen SQL and data analysis skills.

## 📌 Project Overview

This project is based on a Netflix Movies and TV Shows dataset. The main objective is to perform SQL queries to analyze the data and gain meaningful business insights. It demonstrates the use of SQL for data cleaning, filtering, aggregation, joins, window functions, and data analysis.

## 🚀 Objectives

* Analyze Netflix content using SQL.
* Practice real-world SQL interview questions.
* Improve data querying and analytical skills.
* Generate business insights from the dataset.

## 🛠️ Technologies Used

* SQL (MySQL)
* MySQL Workbench
* Netflix Dataset (CSV)

## 📊 Key SQL Concepts Covered

* SELECT, WHERE, ORDER BY
* GROUP BY & HAVING
* Aggregate Functions
* CASE Statements
* Common Table Expressions (CTEs)
* Window Functions
* String & Date Functions
* Data Cleaning
* Subqueries
* Views

## 📈 Business Questions Solved

* Total number of Movies and TV Shows.
* Top countries producing Netflix content.
* Most common content ratings.
* Movies released each year.
* Top directors with the highest number of titles.
* Longest movies on Netflix.
* Most popular genres.
* Content added to Netflix over time.
* Duration analysis of movies and TV shows.
* Year-wise content distribution.
